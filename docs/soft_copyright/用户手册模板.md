## 1. 软件介绍

### 1.1 产品概述

AI 短剧制作系统是一款基于人工智能技术的短剧自动化制作软件。系统整合了 AI 大模型在文本、图像、音频、视频领域的生成能力，实现从剧本创作到成品输出的全流程自动化。

### 1.2 核心功能

- **AI 剧本生成**: 输入主题即可自动生成完整短剧剧本
- **智能配音**: 多音色选择，文本自动转语音
- **视频合成**: 根据剧本自动匹配画面，生成完整视频
- **素材管理**: 海量音视频素材库，支持分类检索
- **会员计费**: 支持会员套餐、AI 点数等多种计费模式

### 1.3 适用场景

- 个人短剧创作者
- 自媒体博主
- 中小影视工作室
- 电商短视频团队

## 3. 安装部署

### 3.1 环境准备

```bash
# 安装 Nginx
sudo apt install nginx

# 安装 PHP 8.0
sudo apt install php8.0 php8.0-fpm php8.0-mysql php8.0-redis

# 安装 MySQL
sudo apt install mysql-server

# 安装 Redis
sudo apt install redis-server
```

### 3.2 代码部署

```bash
# 1. 上传代码
cd /var/www
git clone https://github.com/yourname/ai-drama-system.git

# 2. 安装依赖
cd ai-drama-system/backend
composer install

# 3. 配置数据库
mysql -u root -p
CREATE DATABASE ai_drama;
exit

# 4. 导入数据
mysql -u root -p ai_drama < docs/database/schema.sql

# 5. 配置环境变量
cp .env.example .env
vim .env  # 编辑数据库连接信息
```

### 3.3 Nginx 配置

```nginx
server {
    listen 80;
    server_name yourdomain.com;
    root /var/www/ai-drama-system/backend/public;
    index index.php;

    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    location ~ \.php$ {
        fastcgi_pass unix:/run/php/php8.0-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }
}
```

### 3.4 启动服务

```bash
# 重启 Nginx
sudo systemctl restart nginx

# 重启 PHP-FPM
sudo systemctl restart php8.0-fpm

# 启动 Redis
sudo systemctl start redis
```

## 4. 功能说明

### 4.1 用户注册登录

**功能描述**: 用户可以通过手机号验证码或微信授权进行注册登录。

**操作步骤**:
1. 打开应用首页
2. 点击"登录/注册"按钮
3. 输入手机号，获取验证码
4. 输入验证码，完成登录

### 4.2 AI 剧本生成

**功能描述**: 用户输入短剧类型、主题、时长等参数，系统自动生成完整剧本。

**操作步骤**:
1. 进入"创作"页面
2. 选择"AI 写剧本"
3. 选择短剧类型（情感/反转/搞笑/职场/电商）
4. 输入主题和风格
5. 点击"立即生成"
6. 等待 AI 生成完成（约 30 秒）
7. 预览并编辑剧本

### 4.3 智能配音

**功能描述**: 将剧本文字转换为语音，支持多音色选择。

**操作步骤**:
1. 进入"创作"页面
2. 选择"AI 配音"
3. 输入或粘贴文本内容
4. 选择音色（男声/女声/童声等）
5. 调节语速
6. 点击"生成配音"
7. 试听并保存

### 4.4 视频合成

**功能描述**: 根据剧本和配音，自动匹配画面生成完整视频。

**操作步骤**:
1. 进入"创作"页面
2. 选择"AI 成片"
3. 选择已生成的剧本
4. 选择配音文件
5. 选择背景音乐
6. 点击"生成视频"
7. 等待合成完成（约 2-5 分钟）

### 4.5 作品管理

**功能描述**: 查看、编辑、导出、分享已创作的短剧作品。

**操作步骤**:
1. 进入"作品"页面
2. 浏览作品列表
3. 点击作品查看详情
4. 可进行导出、分享、删除操作

### 4.6 会员中心

**功能描述**: 开通会员、充值 AI 点数、查看订单。

**操作步骤**:
1. 进入"我的"页面
2. 点击"开通会员"
3. 选择套餐（月度/季度/年度）
4. 选择支付方式
5. 完成支付
