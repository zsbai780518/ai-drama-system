## 🎯 使用指南

### 第一步：阅读申请指南
📄 阅读文件：`软件著作权申请材料.md`
- 了解完整申请流程
- 确认申请类型（个人/企业）
- 准备基本信息

### 第二步：准备材料
📄 参考文件：`README.md` + `快速检查清单.md`
- [ ] 身份证明扫描件
- [ ] 运行源代码生成脚本
- [ ] 准备用户手册（含截图）
- [ ] 填写申请表

### 第三步：生成源代码 PDF
```bash
cd /home/admin/openclaw/workspace/ai-drama-system/docs/soft_copyright
python3 generate_source_code_pdf.py
```
输出：
- `source_code_front.pdf` (前端代码 30 页)
- `source_code_back.pdf` (后端代码 30 页)

### 第四步：编写用户手册
📄 参考文件：`用户手册模板.md`
- 复制模板
- 填充实际内容
- 添加运行截图（6 张以上）
- 导出为 PDF

### 第五步：在线提交
1. 访问 https://register.ccopyright.com.cn
2. 注册/登录账号
3. 填写申请表（参考 `申请表填写模板.docx`）
4. 上传所有材料
5. 缴纳费用
6. 提交申请
